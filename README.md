<p align="center">
Large-scale evaluation of self-supervised learning for smartphone-based HAR using the DAGHAR benchmark dataset. 11,232 models were evaluated, across four SSL techniques, six encoders, two refinement strategies, and six datasets. We show that SSL can outperform supervised learning with fewer labeled samples.
